# 3 分鐘操作示範腳本

> 目標:展示「AI 產生 → 貼上 → 簡報 → 換 template → 分享 → 匯出」全流程 < 3 分鐘。

**0:00 — 開場(15s)**
打開 SnapDeck。左邊是貼上點(不是寫作點),右邊即時 preview。
「假設你有一段會議口述、或剛請 LLM 整理完的素材。」

**0:15 — AI 產生(25s)**
按「✦ AI 產生」複製 prompt,貼進任何 LLM(換上你的素材)。模型回傳
一個可一鍵複製的 Markdown 區塊,整段貼進左欄——外層 code fence 會被
自動剝掉。沒有 LLM 也行:頂欄「載入範例…」選 `03-sales-quarterly`。

**0:40 — 看排版(30s)**
右欄立刻長出投影片:frontmatter 變 title 頁;「**粗體詞**:說明」×3
自動變卡片;`61%,會員貢獻營收,連四季上升` 三欄位格式變大數字頁;
連續幾段數字短句排成 KPI 看板 —— 全程零標記。右上「✓ Lint」:
這份文件通過 profile linter。

**1:10 — 簡報(30s)**
按「▶ 簡報」進全螢幕。方向鍵翻頁,`s` 打開 speaker notes 面板
(講稿藏在 `<!-- notes: -->` 註解裡,GitHub 上看不見)。
底部「⛓ 分享此頁」:對方開連結直接進簡報模式這一頁。Esc 退出。

**1:40 — 換 template(20s)**
preview 頂欄的五個色點一鍵切換(Clean Light / Midnight / Craft /
Forest / Boardroom)。整份 deck 變色,連 mermaid 圖表配色都跟著換
—— 顏色全部來自 template config,內容一個字沒動。

**2:00 — 閱讀頁與分享(30s)**
preview 切「頁面」→「⤢ 全版閱讀」:blog 式閱讀版,附錄(skip 頁)
看得到但不進簡報。按「⛓ 分享」選「閱讀頁連結」——內容壓縮在網址裡
(不經任何伺服器),對方打開就是這個閱讀頁。

**2:30 — 匯出(30s)**
「↓ pptx」:PowerPoint 打開全是**原生文字框**,主管要改兩個字就讓他改;
mermaid 是高解析 PNG,講稿在備忘稿欄位。「↓ HTML」:單一自含檔案,
寄信或丟任何靜態空間都能開。

**收尾一句話**
「寫作即排版,貼上即上台。文件本體始終是一份正常的 Markdown。」
